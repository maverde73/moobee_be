TABELLA: employee_roles
DESCRIZIONE: This model contains an expression index which requires additional setup for migrations. Visit https://pris.ly/d/expression-indexes for more info.

CAMPI UTILIZZABILI (USA ESATTAMENTE QUESTI NOMI):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

IDENTIFICATIVI:
- id (integer): Campo id [PRIMARY KEY]

RELAZIONI PRINCIPALI:
- employee_id (integer): Campo employee_id [REQUIRED] [FK verso employees]
- tenant_id (text): Campo tenant_id [REQUIRED] [FK verso tenants]

ALTRI CAMPI:
- role_id (integer): Campo role_id [REQUIRED]
- sub_role_id (integer): Campo sub_role_id
- anni_esperienza (integer): Campo anni_esperienza [DEFAULT: 0]
- seniority (character varying(50)): Campo seniority

DATE E TIMESTAMP:
- created_at (timestamp without time zone): Campo created_at [DEFAULT: CURRENT_TIMESTAMP]

STATI E FLAGS:
- is_current (boolean): Campo is_current [DEFAULT: false]

RELAZIONI:
━━━━━━━━━━
- employees (N:1): Collegamento via employee_id
  * ON DELETE: RESTRICT
  * ON UPDATE: CASCADE
- tenants (N:1): Collegamento via tenant_id
  * ON DELETE: CASCADE
  * ON UPDATE: NO ACTION

REGOLE BUSINESS:
━━━━━━━━━━━━━━━━
1. Campi obbligatori: employee_id, role_id, tenant_id
2. La cancellazione di tenants cancella automaticamente questo record
3. Il timestamp di creazione viene impostato automaticamente

QUERY COMUNI:
━━━━━━━━━━━━

1. Recupera tutti i record di employee_roles per un tenant:
{
  "table": "employee_roles",
  "select": [
    "id",
    "employee_id",
    "role_id",
    "sub_role_id",
    "created_at"
  ],
  "where": {
    "tenant_id": "uuid_tenant"
  },
  "limit": 100
}

2. Recupera employee_roles con dati di employees:
{
  "table": "employee_roles",
  "select": [
    "*"
  ],
  "leftJoin": [
    {
      "table": "employees",
      "first": "employee_roles.employee_id",
      "second": "employees.id"
    }
  ],
  "limit": 50
}

3. Conta employee_roles creati nell'ultimo mese:
{
  "table": "employee_roles",
  "select": [
    "COUNT(*) as total"
  ],
  "where": {
    "created_at": {
      "operator": ">=",
      "value": "NOW() - INTERVAL '30 days'"
    }
  }
}
