QUERY COMUNI:
━━━━━━━━━━━━

1. Recupera tutti i record di soft_skills_assessments per un tenant:
{
  "table": "soft_skills_assessments",
  "select": [
    "id",
    "assessment_id",
    "employee_id",
    "communication_score",
    "teamwork_score"
  ],
  "where": {
    "tenant_id": "uuid_tenant"
  },
  "limit": 100
}

2. Conta soft_skills_assessments creati nell'ultimo mese:
{
  "table": "soft_skills_assessments",
  "select": [
    "COUNT(*) as total"
  ],
  "where": {
    "created_at": {
      "operator": ">=",
      "value": "NOW() - INTERVAL '30 days'"
    }
  }
}
