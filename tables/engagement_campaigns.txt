IDENTIFICATIVI:

ALTRI CAMPI:

- target_audience (jsonb): Campo target_audience [JSONB]

DATI ANAGRAFICI:
- name (text): Campo name [REQUIRED]

DATI DESCRITTIVI:
- description (text): Campo description

DATE E TIMESTAMP:
- start_date (timestamp without time zone): Campo start_date [REQUIRED]
- end_date (timestamp without time zone): Campo end_date [REQUIRED]
- created_at (timestamp without time zone): Campo created_at [REQUIRED] [DEFAULT: CURRENT_TIMESTAMP]
- updated_at (timestamp without time zone): Campo updated_at [REQUIRED]

STATI E FLAGS:
- status (text): Campo status [REQUIRED] [DEFAULT: PLANNED]
- has_responses (boolean): Campo has_responses [REQUIRED] [DEFAULT: false]

QUERY COMUNI:
━━━━━━━━━━━━

1. Recupera tutti i record di engagement_campaigns per un tenant:
{
  "table": "engagement_campaigns",
  "select": [
    "id",
    "tenant_id",
    "template_id",
    "name",
    "description"
  ],
  "where": {
    "tenant_id": "uuid_tenant"
  },
  "limit": 100
}

2. Recupera engagement_campaigns con dati di engagement_templates:
{
  "table": "engagement_campaigns",
  "select": [
    "*"
  ],
  "leftJoin": [
    {
      "table": "engagement_templates",
      "first": "engagement_campaigns.template_id",
      "second": "engagement_templates.id"
    }
  ],
  "limit": 50
}

3. Ricerca engagement_campaigns per name:
{
  "table": "engagement_campaigns",
  "select": [
    "*"
  ],
  "where": {
    "name": {
      "operator": "like",
      "value": "%search%"
    }
  },
  "limit": 20
}

4. Conta engagement_campaigns creati nell'ultimo mese:
{
  "table": "engagement_campaigns",
  "select": [
    "COUNT(*) as total"
  ],
  "where": {
    "created_at": {
      "operator": ">=",
      "value": "NOW() - INTERVAL '30 days'"
    }
  }
}
