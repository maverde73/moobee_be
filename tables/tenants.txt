QUERY COMUNI:
━━━━━━━━━━━━

1. Recupera tutti i record di tenants:
{
  "table": "tenants",
  "select": [
    "id",
    "slug",
    "name",
    "companyName",
    "domain"
  ],
  "where": {
    "is_active": true
  },
  "limit": 100
}

2. Ricerca tenants per name:
{
  "table": "tenants",
  "select": [
    "*"
  ],
  "where": {
    "name": {
      "operator": "like",
      "value": "%search%"
    }
  },
  "limit": 20
}
