QUERY COMUNI:
━━━━━━━━━━━━

1. Recupera tutti i record di engagement_surveys per un tenant:
{
  "table": "engagement_surveys",
  "select": [
    "id",
    "employee_id",
    "survey_month",
    "job_satisfaction",
    "work_life_balance"
  ],
  "where": {
    "tenant_id": "uuid_tenant"
  },
  "limit": 100
}

2. Conta engagement_surveys creati nell'ultimo mese:
{
  "table": "engagement_surveys",
  "select": [
    "COUNT(*) as total"
  ],
  "where": {
    "created_at": {
      "operator": ">=",
      "value": "NOW() - INTERVAL '30 days'"
    }
  }
}
