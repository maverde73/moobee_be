QUERY COMUNI:
━━━━━━━━━━━━

1. Recupera tutti i record di engagement_templates per un tenant:
{
  "table": "engagement_templates",
  "select": [
    "id",
    "title",
    "description",
    "type",
    "category"
  ],
  "where": {
    "tenant_id": "uuid_tenant"
  },
  "limit": 100
}

2. Ricerca engagement_templates per title:
{
  "table": "engagement_templates",
  "select": [
    "*"
  ],
  "where": {
    "title": {
      "operator": "like",
      "value": "%search%"
    }
  },
  "limit": 20
}

3. Conta engagement_templates creati nell'ultimo mese:
{
  "table": "engagement_templates",
  "select": [
    "COUNT(*) as total"
  ],
  "where": {
    "created_at": {
      "operator": ">=",
      "value": "NOW() - INTERVAL '30 days'"
    }
  }
}
